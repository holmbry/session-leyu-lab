# session-leyu-lab

本仓库用于归档和发布多个独立的 HTML 页面。这些页面通常为一次性或长期维护的静态页面，不针对某一特定网站或域名。

## 项目简介

`session-leyu-lab` 是一个以 HTML 页面为主的静态资源仓库。仓库中的每个页面独立于其他项目运行，可用于测试、展示、存档或其他临时用途。页面不依赖后端服务，可直接在浏览器中打开。

## 目录说明

```
/
├── index.html          # 仓库入口页面（可选）
├── page-xxx/           # 按功能或用途命名的文件夹
│   └── index.html
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- 每个子文件夹对应一组相关的 HTML 页面。
- 公共资源存放在 `assets/` 目录下，供各页面引用。
- 根目录下的 `index.html` 可作为导航页，列出所有归档页面。

## 页面归档说明

- 所有页面均为纯 HTML、CSS 和 JavaScript 实现。
- 页面之间相互独立，无数据或状态共享。
- 部分页面可能包含外部资源（如 CDN 库），请确保网络可用。
- 归档页面可能不定期更新或移除，恕不另行通知。
